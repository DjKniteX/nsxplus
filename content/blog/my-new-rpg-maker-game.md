I am back on my RPG Maker grind. Been working on a short RPG with roguelite features
with the working title being "TSN Rogue". Right now the first demo is using the stock RTP so the game looks like a generic RPG Maker game. The second demo will include the same RTP but will have more features and be much more playable than the fist demo. The demo can be played on my itch.io

Once the second demo is out, I will begin working on 0.5 which will end with close friends and family alpha something around 0.4. The alpha will include custom graphics and a much more refined system and gameplay loop. After the Alpha, I will begin work on porting it over to MZ now that MV is over a decade old. Or maybe parallel if MZ goes on sale for a good price.

Some of the things I will be working on for the 2nd demo are:

- 2 New Dungeons
- Changes to old Dungeons
- Secret Boss
- More replayability (Challenge, ascension type runs)
- Working chests
- 2 new characters, one is unlocked and the other is unlockable
- Working TSN Shop (Buy upgrades after each run)
- Save System
- Balanced Stats
- More monsters
- Bug fixes

I bought a notebook so I can keep track of the gameplay loop. But right now the game is using stock maps and RTP and hoping to change that in the near future. No estimated date for the 2nd demo but I'm hoping by the summer. After the Friends and Family Alpha, I will make a Steam Alpha release as a demo for 0.5, and then possibily thinking either sell the game for $5 starting or free at 1.0. We will see!
